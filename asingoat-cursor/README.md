# Gradient Mesh Wallpaper Creator

Una aplicación web React para crear mallas de gradiente personalizadas para wallpapers de computadora.

## Características

- **Puntos de color arrastrables**: Agrega puntos de color haciendo clic en el canvas y arrastra para reposicionarlos
- **Selector de color**: Haz clic en cualquier punto de color para cambiar su color
- **Exportación PNG**: Exporta tu creación como imagen PNG en diferentes resoluciones (Full HD, QHD, 4K)
- **Vista previa en tiempo real**: Ve los cambios instantáneamente mientras trabajas
- **Interfaz moderna**: Diseño limpio siguiendo el sistema de diseño de Mercado Libre

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Construcción

```bash
npm run build
```

## Uso

1. Haz clic en el canvas para agregar un nuevo punto de color
2. Arrastra los puntos de color para reposicionarlos
3. Haz clic en un punto de color para cambiar su color usando el selector
4. Haz clic derecho en un punto para eliminarlo
5. Selecciona la resolución de exportación deseada
6. Haz clic en "Exportar como PNG" para descargar tu wallpaper

## Tecnologías

- React 18
- Vite
- Canvas API
- Sistema de diseño Mercado Libre
